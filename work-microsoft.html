PM/SE Intern @ Microsoft

Team Windows Experimentation
Location Redmond, WA
Duration Summer 2017

Last summer, I had a really unique opportunity to experience both product management and software engineering roles within a single internship (called the Explorer program). Although the Windows operating system is officially released once every couple of years, there exists an Insider Program of over 15 million people who can access the latest Windows releases every week. I focused on building tools to help manage this release process as well as improvements to be able to effectively experiment and A/B test on these releases.

It was amazing to see that even as an intern, I could have an impact at such a large scale.

A few highlights:

Engineered a tool to assess & segment activity on 70,000+ devices. Soft launched with 20 engineers, achieving a 60% increase in satisfaction.

Designed automated feedback surveys for the Windows OS. Reduced survey creation time by 7x.

Devised automated attribute manager to target & collect data from Windows devices, unlocking 8 hours of developer time per week.
